# MPB

MPB is a modular diy power bank that is 100% scalable it does have batterie protection leds to show charging status 
Keep in mind that this project is still work in progress it lacks some ideas 

# BOM ( Bill of Material )
- USB C receptacle
- two Leds 
- ten 1x02 pinHeaders
- two Nmos transistors
- five capacitors
- a tp4056
- DW0A1
-  one voltage regulator BD33FC0WFP
-  ten resistors
